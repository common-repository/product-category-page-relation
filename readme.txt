== Description ==



This plugin is usefull for setup a woocommerce category page custom template , you have option to add content on product category to a contnet of page . 

 



== Installation ==

 



1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. edit your product category and add there product slug
